## B. AI/LLM 활용 범위 (Opus 사용 가능 시)

### B-1. LLM 역할 선택 (복수 선택 가능)

비용 제약이 없어진 상황에서 LLM을 어디에 쓸지 결정:

- [ ] **B-1a. Reddit/소셜 감성 압축** (근거 있음, 추천)
  - 매 4시간 Reddit 크립토 서브레딧 → 감성 점수 → 매매 거부권/사이즈 배수
  - 모델: Haiku/Sonnet 충분
- [ ] **B-1b. 뉴스/거시경제 내러티브 분석** (증거 약함, 실험적)
  - FOMC 성명, CPI 발표, 한국 규제 뉴스를 Opus가 읽고 리스크 스코어 생성
  - 주의: 뉴스 감성은 Reddit보다 성능 낮음 (연구 결과)
- [ ] **B-1c. 레짐 내러티브 해석** (보조 역할, 로깅용)
  - 룰 기반 레짐 분류기 결과를 Opus가 자연어로 해설 → 대시보드/알림에 표시
  - 매매 결정에는 영향 없음
- [ ] **B-1d. 다중 에이전트 아키텍처 (TradingAgents 스타일)** (실험적, 주의)
  - 분석가/연구원/리스크매니저/트레이더 역할의 Opus 에이전트들이 토론
  - 주의: 크립토 라이브 검증 없음. 주식 불장 백테스트만 존재. 연구 프로젝트 성격.
- [ ] **B-1e. 직접 매매 결정 (Opus가 최종 "매수/매도/홀드")** (비추천)
  - 주의: Agent Market Arena(2025) 라이브에서 프론티어 LLM 실패 증명
- [ ] **B-1f. 백테스트 결과 사후 분석 리포트 생성** (안전, 유용)
  - 매일/매주 백테스트 및 라이브 성과를 Opus가 자연어 리포트로 생성
  - 패턴, 실패 원인, 개선 제안
- [ ] **B-1g. 코인 화이트리스트 주간 리뷰** (유용)
  - 매주 일요일 Opus가 각 코인의 최근 뉴스/온체인/거래량을 종합해 매매 유니버스 갱신

**선택한 항목 번호:**
너 생각에 어디어디에 쓰면 좋을거같은지 확인해보고 알려줘
