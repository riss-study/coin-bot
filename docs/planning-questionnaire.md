## 1. 거래소 & 자산

| # | 질문 | 답변 |
|---|------|------|
| 1-1 | 어떤 거래소를 사용할 건가요? (예: 업비트, 바이낸스, 비트겟, 복수 선택 가능) | | 업비트, 코인원 중에면 좋은데, 직접 자동매매할 수 있으려면 이걸 api 로 하든지 프로그램이 직접해야 하는데 이 연동이 가능한 플랫폼인지 체크부터 해야할 듯해.
| 1-2 | 거래 대상 코인은? 특정 코인만? 시가총액 상위 N개 자동? 전체? | | 전체 지만 스캠코인은 무조건 제외. 변동에 대한 예측이 불가능한 코인보다 예측이 가능한 코인들 위주면 좋을 듯 (이건 너의 판단에 맡김)
| 1-3 | 현물(Spot) 거래만? 선물(Futures)/레버리지도 포함? | | 선물같이 위험한 건 지양
| 1-4 | 투자 원금 규모는 대략 어느 정도? (봇 설계 시 수수료·슬리피지 전략이 달라짐) | | 일단 10만원~100만원부터 시작.

## 2. 매매 전략 & 알고리즘

| # | 질문 | 답변 |
|---|------|------|
| 2-1 | 선호하는 매매 스타일은? (스캘핑 / 단타·데이트레이딩 / 스윙 / 복합) | | 이건 잘 모르겠음. 추천해줘. 10번의 이상의 검증을 통해.
| 2-2 | 기술적 지표(TA) 중 선호하는 게 있나요? (RSI, MACD, 볼린저밴드, 이동평균선 등) | | 잘 모름. 연구 엄청 해봐봐. 가장 신뢰되는 지표 혹은 복합 지표 도 괜찮으면 직접 연구해줘. 최소 10번 이상의 검증 필요.
| 2-3 | 손절/익절 기준은? (예: -3% 손절, +5% 익절 같은 고정 비율? 동적 조절?) | | 이것도 너의 연구를 통해 검증해줘. 10번 이상.
| 2-4 | DCA(분할매수) 전략을 포함할 건가요? | | 분할매수도 괜찮음
| 2-5 | 동시에 몇 개 코인까지 포지션을 잡을 건가요? | | 너의 10번 이상의 검증을 통한 적정 수를 찾아줘.
